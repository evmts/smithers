# Git Automation with Ralph

This template demonstrates Ralph's git automation capabilities including:
- Automated code reviews with Codex
- Git notes for conversation context
- Multi-phase workflow

<Claude>

    </Step>
  </Phase>

  <Phase name="recommend">
    <Step>

      - Good practices for commit messages in this project

    </Step>
  </Phase>

  <Phase name="demo">
    <Step>

    </Step>
  </Phase>
