# ClaudeApi

```tsx
<ClaudeApi model="claude-sonnet-4">
  Generate a haiku about programming.
</ClaudeApi>
```

Direct Anthropic SDK access, bypassing Claude Code CLI. Use for simple completions where full tool access is unnecessary.

## Props

| Prop | Type | Description |
|------|------|-------------|
| `model` | `string` | Anthropic model ID (e.g., "claude-sonnet-4") |
| `maxTurns` | `number` | Maximum conversation turns |
| `tools` | `string[]` | Tool names to make available |
| `systemPrompt` | `string` | System prompt |
| `onFinished` | `(result: ClaudeApiResult) => void` | Success callback |
| `onError` | `(error: Error) => void` | Error callback |
| `children` | `ReactNode` | Prompt to send |

## ClaudeApiResult

```typescript
interface ClaudeApiResult<T = unknown> {
  output: string
  structured?: T
  tokensUsed: { input: number; output: number }
  turnsUsed: number
  stopReason: 'completed' | 'stop_condition' | 'error' | 'cancelled'
  durationMs: number
}
```

## Claude vs ClaudeApi

| Feature | Claude | ClaudeApi |
|---------|--------|-----------|
| Tool access | Full CLI tools | Custom only |
| File editing | Built-in | Manual |
| Database logging | Automatic | Manual |
| Best for | Agentic tasks | Simple completions |

## Environment

```bash
export ANTHROPIC_API_KEY="your-api-key"
```

<Warning>
Unlike `<Claude>` which uses Claude Code's subscription, ClaudeApi bills to your Anthropic account.
</Warning>
