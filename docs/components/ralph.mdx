# Ralph

<Warning>
**Deprecated.** Use `<SmithersProvider>` which handles Ralph internally. This page is for advanced users.
</Warning>

```tsx
<SmithersProvider db={db} executionId={executionId}>
  <Ralph>
    <Claude>Your prompt here</Claude>
  </Ralph>
</SmithersProvider>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `maxIterations` | `number` | Deprecated. Use `SmithersProvider` prop |
| `onIteration` | `(n: number) => void` | Deprecated. Use `SmithersProvider` prop |
| `onComplete` | `() => void` | Deprecated. Use `SmithersProvider` prop |
| `onMaxIterations` | `() => void` | Called when max iterations reached |

## Accessing Iteration Count

```tsx
function MyComponent() {
  const { ralphCount } = useSmithers();
  return <div>Iteration: {ralphCount}</div>;
}
```
