# Notes

```tsx
import { Notes } from "smithers-orchestrator";

<Notes
  data={{
    reviewedBy: "smithers",
    confidence: 0.95,
    issues: []
  }}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `commitRef` | `string` | `"HEAD"` | Git commit reference |
| `data` | `Record<string, any>` | required | Structured metadata |
| `append` | `boolean` | `false` | Append vs replace existing notes |
| `onFinished` | `(result: NotesResult) => void` | - | Success callback |
| `onError` | `(error: Error) => void` | - | Error callback |

## NotesResult

```typescript
interface NotesResult {
  commitRef: string;
  data: Record<string, any>;
  previousNotes: string | null;
}
```

## Automatic Metadata

Notes include smithers tracking:

```json
{
  "smithers": true,
  "executionId": "exec-123",
  "timestamp": 1705596000000,
  "reviewedBy": "smithers"
}
```

## With Commit

```tsx
<Git.Commit message="feat: add auth">
  <Notes data={{ reviewPassed: true, testsRun: 42 }} />
</Git.Commit>
```
