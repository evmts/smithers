# Output Component

The `<Output>` component renders content to the terminal during execution or changes the final rendered output. It's useful for displaying progress, results, or status messages without requiring a Claude execution.

## Basic Usage

```tsx
import { Output } from '@evmts/smithers'

<Output>
  Processing complete! Found 42 files.
</Output>
```

## Props

<ParamField path="children" type="JSX.Element" required>
  The content to output. Can be plain text, markdown, or nested components.
</ParamField>

<ParamField path="format" type="'text' | 'json' | 'markdown'" default="text">
  The format of the output content.

  ```tsx
  <Output format="json">
    {JSON.stringify({ status: 'complete', count: 42 })}
  </Output>
  ```
</ParamField>

<ParamField path="label" type="string">
  Optional label to prefix the output.

  ```tsx
  <Output label="Result">
    Analysis complete
  </Output>
  // Renders: [Result] Analysis complete
  ```
</ParamField>

## Use Cases

### Displaying Results

Show results between agent phases:

```tsx
import { createSignal } from 'solid-js'

function AnalysisAgent(props) {
  const [result, setResult] = createSignal(null)

  return () => {
    const r = result()
    if (!r) {
      return (
        <Claude onFinished={setResult} tools={[filesystem]}>
          Analyze {props.files.length} files for issues
        </Claude>
      )
    }

    return (
      <>
        <Output format="markdown">
          ## Analysis Complete

          Found {r.issues.length} issues across {props.files.length} files.

          ### Summary
          - Critical: {r.critical}
          - Warnings: {r.warnings}
        </Output>
        <Stop reason="Analysis complete" />
      </>
    )
  }
}
```

### Progress Updates

Show progress during multi-phase workflows:

```tsx
import { createSignal } from 'solid-js'

function MultiPhaseAgent() {
  const [phase, setPhase] = createSignal('research')
  const [data, setData] = createSignal(null)

  return () => {
    const p = phase()
    const d = data()

    if (p === 'research') {
      return (
        <Claude onFinished={(res) => { setData(res); setPhase('report') }}>
          Research the topic
        </Claude>
      )
    }

    if (p === 'report') {
      return (
        <>
          <Output label="Phase 1">Research complete, generating report...</Output>
          <Claude onFinished={() => setPhase('done')}>
            Generate report from: {JSON.stringify(d)}
          </Claude>
        </>
      )
    }

    return <Output>All phases complete!</Output>
  }
}
```

### JSON Output

Output structured data:

```tsx
<Output format="json">
  {JSON.stringify({
    status: 'success',
    files_processed: 42,
    errors: [],
    timestamp: new Date().toISOString()
  }, null, 2)}
</Output>
```

### Conditional Output

Show output based on conditions:

```tsx
import { createSignal } from 'solid-js'

function StatusAgent(props) {
  const [status, setStatus] = createSignal(null)

  return () => {
    const s = status()
    return (
      <>
        <Claude onFinished={setStatus}>Check system status</Claude>

        {s && (
          <>
            <Output>Status: {s.health}</Output>
            {props.verbose && (
              <Output format="json">{JSON.stringify(s.details)}</Output>
            )}
          </>
        )}
      </>
    )
  }
}
```

## Combining with Other Components

Output works well with control flow components:

```tsx
import { createSignal } from 'solid-js'
import { createStore } from 'solid-js/store'

function WorkflowAgent(props) {
  const [step, setStep] = createSignal(1)
  const [results, setResults] = createStore({})

  return () => {
    const s = step()

    if (s === 1) {
      return (
        <Claude onFinished={(r) => { setResults('step1', r); setStep(2) }}>
          Analyze: {props.task}
        </Claude>
      )
    }

    if (s === 2) {
      return (
        <>
          <Output label="Step 1 Complete">
            Analysis found {results.step1?.issues || 0} issues
          </Output>
          <Human
            message="Continue to step 2?"
            onApprove={() => setStep(3)}
          >
            {JSON.stringify(results.step1, null, 2)}
          </Human>
        </>
      )
    }

    if (s === 3) {
      return (
        <Claude onFinished={(r) => { setResults('step2', r); setStep(4) }}>
          Fix the issues from: {JSON.stringify(results.step1)}
        </Claude>
      )
    }

    return (
      <>
        <Output format="markdown">
          ## Workflow Complete

          - Step 1: {results.step1?.summary}
          - Step 2: {results.step2?.summary}
        </Output>
        <Stop reason="Workflow complete" />
      </>
    )
  }
}
```

## Behavior

- Output components are rendered but do not trigger Claude execution
- Multiple Output components are displayed in order
- Output content is included in the serialized XML plan
- In CLI mode, output is printed to stdout
