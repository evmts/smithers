# Task

```tsx
<Phase name="implementation">
  <Step name="tasks">
    <Task done={false}>Research existing patterns</Task>
    <Task done={true}>Design component API</Task>
    <Task done={false}>Implement component</Task>
  </Step>
</Phase>
```

Renders a checkbox-style task item for tracking progress through workflows.

<Note>
**Not to be confused with `db.tasks.*`**: that's the internal API for tracking async component lifecycle. This `<Task>` component is purely presentational.
</Note>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `done` | `boolean` | `false` | Whether the task is complete |
| `children` | `ReactNode` | - | Task description |

## Dynamic Task Tracking

```tsx
function TaskTracker() {
  const { reactiveDb } = useSmithers()
  const { data: tasks } = useQueryValue<string>(
    reactiveDb, "SELECT value FROM state WHERE key = 'tasks'"
  )
  const parsed = tasks ? JSON.parse(tasks) : []

  return (
    <Step name="tasks">
      {parsed.map((task, i) => (
        <Task key={i} done={task.done}>{task.name}</Task>
      ))}
    </Step>
  )
}
```

## Task-Driven Workflow

```tsx
const tasks = ["Analyze", "Refactor", "Test", "Document"]

function TaskDrivenWorkflow() {
  const { db, reactiveDb } = useSmithers()
  const { data: indexVal } = useQueryValue<string>(
    reactiveDb, "SELECT value FROM state WHERE key = 'idx'"
  )
  const idx = indexVal ? JSON.parse(indexVal) : 0

  return (
    <>
      <Step name="tasks">
        {tasks.map((t, i) => <Task key={i} done={i < idx}>{t}</Task>)}
      </Step>
      <If condition={idx < tasks.length}>
        <Claude onFinished={() => db.state.set('idx', idx + 1)}>
          Complete: {tasks[idx]}
        </Claude>
      </If>
    </>
  )
}
```

## Rendering

```xml
<task done="true">Research existing patterns</task>
<task done="false">Implement component</task>
```

## Best Practices

- **Keep tasks atomic**: each task should represent a single action
- **Use with Phase**: group related tasks within phases
