# Sqlite

```tsx
<Claude model="sonnet" maxTurns={5}>
  <Sqlite path="./analytics.db">
    Tables: user_events, sessions
  </Sqlite>
  What are the top 10 most common user actions this week?
</Claude>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `path` | `string` | required | Path to SQLite database file |
| `readOnly` | `boolean` | `false` | Open in read-only mode |
| `children` | `ReactNode?` | - | Schema description for Claude |

## Multiple Databases

```tsx
<Claude model="sonnet" maxTurns={10}>
  <Sqlite path="./users.db">users, profiles, preferences</Sqlite>
  <Sqlite path="./analytics.db">events, sessions, pageviews</Sqlite>
  Find users who signed up but never logged an event.
</Claude>
```
