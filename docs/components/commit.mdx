# Commit

```tsx
<Commit
  message="feat: Add user authentication"
  onFinished={(result) => console.log("Committed:", result.commitHash)}
/>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `message` | `string` | Commit message |
| `autoGenerate` | `boolean` | Auto-generate message from changes |
| `files` | `string[]` | Specific files to commit |
| `all` | `boolean` | Stage all changes (`git add -A`) |
| `notes` | `Record<string, any>` | Git notes to attach |
| `onFinished` | `(result: CommitResult) => void` | Called on completion |
| `onError` | `(error: Error) => void` | Called on error |

### CommitResult

```tsx
interface CommitResult {
  commitHash: string;
  message: string;
  filesChanged: number;
  insertions: number;
  deletions: number;
}
```

## Auto-Generated Messages

```tsx
<Commit
  autoGenerate
  all
  notes={{ smithers: true }}
  onFinished={(r) => console.log("Message:", r.message)}
/>
```

## With Git Notes

```tsx
<Commit
  message="feat: Add feature"
  notes={{
    smithers: true,
    executionId: executionId,
    prompt: "Add user authentication",
  }}
/>
```

## Database Tracking

Commits are logged:

```tsx
const commits = await db.vcs.getCommits(10);
// [{ hash: "abc123", message: "feat: Add feature", ... }]
```
