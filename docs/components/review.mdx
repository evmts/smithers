# Review

```tsx
<Review
  target={{ type: "diff", ref: "main" }}
  criteria={["No security vulnerabilities", "Tests cover edge cases"]}
  onFinished={(review) => {
    if (review.approved) console.log("Passed!");
    else console.log("Issues:", review.issues);
  }}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `target` | `ReviewTarget` | required | What to review (see below) |
| `criteria` | `string[]` | `[]` | Review criteria to check |
| `model` | `string` | `'claude-sonnet-4'` | Model to use for review |
| `blocking` | `boolean` | `false` | Block workflow on failure |
| `postToGitHub` | `boolean` | `false` | Post comments to GitHub PR |
| `postToGitNotes` | `boolean` | `false` | Store review in git notes |
| `agent` | `'claude'` | - | Agent to use (currently only 'claude') |
| `onFinished` | `(result: ReviewResult) => void` | - | Called on completion |
| `onError` | `(error: Error) => void` | - | Called on error |

### ReviewTarget

```tsx
{ type: "diff", ref: "main" }       // Diff against branch
{ type: "commit", ref: "abc123" }   // Specific commit
{ type: "pr", ref: "123" }          // Pull request
{ type: "files", files: [...] }     // Specific files
```

### ReviewResult

```tsx
interface ReviewResult {
  approved: boolean;
  summary: string;
  issues: Array<{
    severity: 'critical' | 'major' | 'minor';
    file?: string;
    line?: number;
    message: string;
    suggestion?: string;
  }>;
}
```

## PR Review with GitHub Posting

```tsx
<Review
  target={{ type: "pr", ref: "123" }}
  model="opus"
  criteria={[
    "Changes are backwards compatible",
    "Security considerations addressed",
  ]}
  postToGitHub
  onFinished={(review) => {
    if (!review.approved) process.exit(1);
  }}
/>
```
