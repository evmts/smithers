# WorktreeProvider

```tsx
import { WorktreeProvider, useWorktree } from "smithers-orchestrator";

<WorktreeProvider value={{ cwd: "/path/to/worktree", branch: "feature-x", isWorktree: true }}>
  <MyWorkflow />
</WorktreeProvider>
```

## useWorktree

```tsx
function MyComponent() {
  const worktree = useWorktree();
  if (!worktree) return <text>Not in a worktree</text>;
  
  return (
    <Claude cwd={worktree.cwd}>
      Working on {worktree.branch}
    </Claude>
  );
}
```

## Context Value

| Property | Type | Description |
|----------|------|-------------|
| `cwd` | `string` | Working directory path |
| `branch` | `string` | Git branch name |
| `isWorktree` | `true` | Always true within provider |

## With Parallel Worktrees

```tsx
<Parallel>
  <Worktree branch="feature-a">
    <FeatureWorkflow />
  </Worktree>
  <Worktree branch="feature-b">
    <FeatureWorkflow />
  </Worktree>
</Parallel>

function FeatureWorkflow() {
  const { cwd, branch } = useWorktree()!;
  return <Claude cwd={cwd}>Implement feature on {branch}</Claude>;
}
```
