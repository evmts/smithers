# Gemini

```tsx
<Gemini model="gemini-2.5-pro">
  Analyze this codebase and identify architectural issues.
</Gemini>
```

Wraps Google Gemini API for AI agent capabilities with multi-modal input (text, images, code) and structured output.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Prompt for the agent |
| `model` | `'gemini-2.5-pro' \| 'gemini-2.5-flash' \| 'gemini-1.5-pro'` | `'gemini-2.5-pro'` | Model selection |
| `temperature` | `number` | `1` | Randomness (0-2) |
| `systemPrompt` | `string` | - | System-level instructions |
| `cwd` | `string` | - | Working directory |
| `env` | `Record<string, string>` | - | Environment variables |
| `timeout` | `number` | `300000` | Timeout in ms |
| `schema` | `ZodType` | - | Structured output schema |
| `media` | `Array<{type: 'image' \| 'file', path: string}>` | - | Multi-modal inputs |
| `onFinished` | `(result: AgentResult) => void` | - | Success callback |
| `onError` | `(error: Error) => void` | - | Error callback |

## Multi-Modal Analysis

```tsx
<Gemini
  model="gemini-2.5-pro"
  media={[
    { type: 'image', path: './screenshots/ui-bug.png' },
    { type: 'image', path: './screenshots/expected.png' }
  ]}
>
  Compare these UI screenshots. Identify differences and suggest CSS fixes.
</Gemini>
```

## With Structured Output

```tsx
import { z } from 'zod'

const analysisSchema = z.object({
  issues: z.array(z.object({
    category: z.enum(['scalability', 'maintainability', 'performance', 'security']),
    severity: z.enum(['critical', 'major', 'minor']),
    description: z.string()
  })),
  overallScore: z.number().min(0).max(10)
})

<Gemini model="gemini-2.5-pro" schema={analysisSchema}>
  Analyze codebase architecture
</Gemini>
```

## In Fallback Chain

## Why Gemini?

- **2M token context**: analyze entire codebases
- **Multi-modal**: native image/video/audio understanding
- **Speed**: Gemini Flash competitive on speed, lower cost
