# smithers init

Create a new Smithers project with all the necessary configuration and a starter agent template.

## Synopsis

```bash
smithers init [directory]
```

## Arguments

<ParamField path="directory" type="string" default=".">
  The directory to initialize. Creates the directory if it doesn't exist. Defaults to the current directory.

  ```bash
  smithers init my-agent-project
  smithers init .
  smithers init ~/projects/research-agents
  ```
</ParamField>

## What It Creates

```
my-agent-project/
  agent.mdx        # Starter agent template (based on --template)
  package.json     # Dependencies configured (if missing)
```

### agent.mdx

```mdx
import { Claude } from 'smithers'

# Hello World Agent

<Claude>
  You are a friendly assistant. Say hello and introduce yourself in one sentence.
</Claude>
```

### package.json

`smithers init` creates a minimal `package.json` if one does not exist:

```json
{
  "name": "my-agent-project",
  "version": "0.1.0",
  "type": "module",
  "scripts": {
    "start": "smithers run agent.mdx",
    "plan": "smithers plan agent.mdx"
  },
  "dependencies": {
    "smithers": "^0.1.0",
    "react": "^19.0.0"
  }
}
```

## Examples

### Create in New Directory

```bash
smithers init my-agent
cd my-agent
npm install
npm run dev -- --props '{"target": "example.com"}'
```

### Initialize Current Directory

```bash
mkdir research-agent
cd research-agent
smithers init
npm install
```

### With Immediate Setup

```bash
smithers init security-scanner && cd security-scanner && npm install
```

## Post-Init Steps

After running `smithers init`:

<Steps>
  <Step title="Install dependencies">
    ```bash
    npm install
    # or
    bun install
    ```
  </Step>

  <Step title="Configure API key">
    ```bash
    # Add your API key to a .env file
    echo "ANTHROPIC_API_KEY=sk-ant-..." > .env
    ```
  </Step>

  <Step title="Preview the starter agent">
    ```bash
    npm run plan
    ```
  </Step>

  <Step title="Run the agent">
    ```bash
    npm run start
    ```
  </Step>
</Steps>

## Project Variants

Choose a starter template with `--template`:

```bash
smithers init --template hello-world
smithers init --template research
smithers init --template multi-agent
```
