# Installation

```bash
bun add smithers-orchestrator
```

## Verify

```tsx
#!/usr/bin/env bun
import { createSmithersRoot, createSmithersDB, SmithersProvider, Claude } from "smithers-orchestrator";

const db = createSmithersDB({ path: ".smithers/test" });
const executionId = db.execution.start("Test", "test.tsx");

function Test() {
  return (
    <SmithersProvider db={db} executionId={executionId}>
      <Claude model="haiku" maxTurns={1} onFinished={(r) => console.log(r.output)}>
        Say "Hello from Smithers!"
      </Claude>
    </SmithersProvider>
  );
}

const root = createSmithersRoot();
await root.mount(Test);
db.close();
```

```bash
bun test.tsx
```

## Claude Code Plugin

```bash
/plugin marketplace add evmts/smithers
/plugin install smithers@smithers
```

Now Claude can generate orchestration workflows for you.

## Dependencies

| Required | Version |
|----------|---------|
| [Bun](https://bun.sh) | 1.0+ |
| [Claude Code](https://www.npmjs.com/package/@anthropic-ai/claude-code) | latest |

**Bundled:** react, bun:sqlite, zod

**Optional:** [jj (Jujutsu)](https://github.com/martinvonz/jj) for VCS snapshots

## Configuration

**tsconfig.json:**
```json
{
  "compilerOptions": {
    "jsx": "preserve",
    "jsxImportSource": "smithers-orchestrator"
  }
}
```

**bunfig.toml:**
```toml
[dev]
jsx = "react"
```
