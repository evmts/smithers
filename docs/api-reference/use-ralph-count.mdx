# useRalphCount Hook

Returns the current `ralphCount` from state. Used internally to trigger agent executions.

```tsx
import { useRalphCount } from "smithers-orchestrator";

function ExecutionTrigger() {
  const count = useRalphCount();
  return <text>Execution count: {count}</text>;
}
```

## API

```tsx
function useRalphCount(): number
```

## Return Value

| Type | Description |
|------|-------------|
| `number` | Current ralph count (defaults to 0) |

## Notes

- Requires `SmithersProvider` context.
- Reactively updates when `ralphCount` changes in state table.
- Used by `useAgentRunner` to detect when to start execution.
