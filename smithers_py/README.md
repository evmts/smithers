# SmithersPy

Python components for the Smithers AI orchestration framework.

## Overview

SmithersPy provides Python-based components and utilities for the Smithers multi-agent AI orchestration system. This package enables Python developers to build and integrate with Smithers workflows.

## Installation

```bash
pip install smithers-py
```

## Development Setup

```bash
# Install in development mode with dev dependencies
pip install -e ".[dev]"

# Run tests
pytest

# Format and lint code
ruff format .
ruff check .
```

## Requirements

- Python 3.11 or higher
- Dependencies: pydantic>=2.0, pydantic-ai, aiosqlite
